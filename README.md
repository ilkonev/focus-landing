# Focus — Прокачай свою цифровую осознанность

## О проекте

**Focus** — это персональный ИИ-помощник для управления экранным временем, который превращает контроль над использованием устройств в осознанную игру. Приложение помогает пользователям развивать здоровые цифровые привычки через геймификацию, умную аналитику и мягкую мотивацию без стресса.

### Основные возможности

- **Умная аналитика** — ИИ отслеживает паттерны поведения и помогает укреплять фокус на основе исследований поведенческой психологии
- **Мотивация через игру** — рейтинги, уровни и челленджи, которые мотивируют соблюдать собственные ограничения и добиваться целей
- **Персональная стратегия** — приложение подстраивается под ваш ритм жизни, вы выбираете уровень контроля от подсказок до строгого режима
- **Адаптивная аналитика** — детальная статистика использования приложений с визуализацией прогресса
- **Мягкая мотивация** — естественное ощущение контроля вместо давления

### Результаты

- **–25%** лишнего экранного времени
- **+18%** осознанного времени
- **–25%** импульсивных действий
- **70%** пользователей соблюдают лимиты

## Технологии

Проект построен с использованием современных веб-технологий:

- **Vite** — быстрый сборщик и dev-сервер
- **TypeScript** — типизированный JavaScript
- **React** — библиотека для создания пользовательских интерфейсов
- **shadcn/ui** — компоненты интерфейса
- **Tailwind CSS** — утилитарный CSS-фреймворк
- **React Router** — маршрутизация
- **TanStack Query** — управление состоянием сервера
